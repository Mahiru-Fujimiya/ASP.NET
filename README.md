# 🎬 LawwyFlix - Hệ Thống Streaming Phim Đỉnh Cao
**Author:** Nguyen Minh Phuc (Lawwy)  
**Version:** 4.0 - Final Edition (2026)

## 🚀 Tính năng chính
- **Cấu trúc Layer:** Web API tách biệt (Data, Business, API).
- **Phân quyền Admin:** Quản lý phim, tập phim, thể loại và người dùng.
- **Giao diện Netflix:** 100% Dark Mode, Responsive, hiệu ứng Hover Zoom.
- **Docker Ready:** Triển khai nhanh chóng với Docker Compose.

## 🛠️ Công nghệ sử dụng
- **Backend:** .NET 8.0, Entity Framework Core, JWT Authentication.
- **Frontend:** HTML5, CSS3 (Bootstrap 5), JavaScript (Fetch API).
- **Database:** SQL Server 2022.
- **DevOps:** Docker, GitHub Actions.

## 📦 Cách chạy dự án
1. Clone dự án: `git clone <link-repo>`
2. Chạy Docker: `docker-compose up --build`
3. Truy cập Swagger: `https://localhost:7254/swagger`
4. Xem phim: `https://localhost:7254/index.html`
